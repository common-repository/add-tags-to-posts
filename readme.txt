Add tags to posts

== Description ==

This plugin will search all posts with specific tags, if found, the tags will automatically add to the post.<br />
Feel free to visit <a href="http://www.trueherb.com/">True Herb</a> to report bugs.

== Installation ==


1. Extract delete-all-tags.zip in the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Delete all tags" In "Settings" Menu
4. Follow the on-screen instructions
